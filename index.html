Sobre o site

Desenvolvi esse site usando o bootstrap, Jquery e um framework para as animações. A ideia aqui é demonstrar o máximo das minhas habilidades por isso não poupei esforços na hora de colorir e animar a página. Muitos dos estilos usados aqui foram escritos por mim mesmo usando o css. Explore o site, clique, passe o mouse e veja que a maioria dos elementos contidos aqui possuem animações e/ou efeitos. Com o passar do tempo realizarei mudanças aqui para melhorar o código ou os elementos do site.
Estarei disponibilizando tudo no GitHub.
